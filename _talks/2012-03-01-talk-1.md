Conducted a hands-on session on **Federated Learning using PyTorch** as a contributing faculty member during the Faculty Development Program on *IoT and Intelligent Systems, Next Gen Communications and AI*.

The session focused on introducing participants to the fundamentals of Federated Learning and guiding them through the implementation of a complete federated learning pipeline using PyTorch. Participants explored client-server training, model aggregation, and practical aspects of developing privacy-preserving distributed machine learning systems.

### Topics Covered

- Fundamentals of Federated Learning
- PyTorch implementation of Federated Learning
- Client-server architecture and Federated Averaging (FedAvg)
- Handling non-IID data distributions
- Practical challenges and research directions
